1) Ano: 2014 Banca: INSTITUTO AOC PÓrgão: UFGD Prova: Analista de Tecnologia da Informação
Considere a Linguagem de Programação Java na sua última versão. Sobre declaração de arrays nesta linguagem, assinale a alternativa INCORRETA.

int[] arrayA [] = {{1},{0}};
int[][] arrayB = new int[][];
int arrayC[][] = {{1},{0}};
int[][] arrayD = new int[1][1];
int[][] arrayE = new int[0][1];

2) Ano: 2015 Banca: FGV Órgão: PGE-RO Prova: Analista da Procuradoria - Analista de Sistemas (Desenvolvimento)
Um programador Java precisa utilizar, em seu código, um arranjo dinâmico de números inteiros. A declaração correta para esse arranjo é:

ArrayList(int) arranjo;
ArrayList(Int) arranjo;
ArrayList(Integer) arranjo[];
ArrayList(Int) arranjo[];
ArrayList(Integer) arranjo;

3) Ano: 2017 Banca: FGV Órgão: IBGE Prova: Analista Censitário - Análise de Sistemas - Desenvolvimento de Aplicações
Um programador Java precisa utilizar um array unidimensional dinâmico armazenando números inteiros e decide importar a classe java.util.ArrayList.
A declaração da referência ao array que ele deverá utilizar é:

ArrayList(int) arr;
ArrayList(int) [ ] arr;
ArrayList(int) arr [ ];
ArrayList(Integer) arr [ ];
ArrayList(Integer) arr.

4) Ano: 2015 Banca: COSEAC Órgão: UFF Prova: Analista de Tecnologia da Informação
Em relação ao uso de arrays na linguagem Java, avalie as afirmativas a seguir.

  1. Um array é um grupo de variáveis que contém valores todos do mesmo tipo.
  2. O primeiro elemento em cada array tem um índice um.
  3. Um arraylist é semelhante a um array, mas pode ser dinamicamente redimensionado.

Das afirmativas acima, apenas:

I está correta.
II está correta.
III está correta.
I e II estão corretas.
I e III estão corretas.

5) Ano: 2013 Banca: CESPE Órgão: SEGESP-AL Prova: Perito Criminal - Tecnologia da Informação
Julgue o item seguinte, a respeito de linguagens orientadas a objeto.
De acordo com a linguagem Java, o array é um objeto, portanto, uma variável de instância de array que for declarada, mas não explicitamente inicializada, terá o valor null.
